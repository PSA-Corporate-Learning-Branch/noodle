Product Essentials

A minimal two-module example that shows how to place Noodle note blocks on a course page with auto-save, manual save, and export support.

How to use

  • TypeFill in the note area under any section.
  • Auto-saveSaving happens 2 seconds after you stop typing.
  • ExportUse the floating export button to download all notes as Markdown.

Section 1

Welcome and Overview

Set the scene and capture quick goals for this course.

10 min

Learn how Noodle tracks notes for each section using unique data attributes. Update the title and placeholder text to match your course tone, then drop in the script reference to enable saves and exports.

Section 2

Module: Foundations

Document key concepts as you work through the basics.

15 min

This module outlines a simple course structure: overview, three lessons, and a quick recap. Swap in your own headings and prompts, and Noodle keeps each section's notes separated by data-sectionid.

Section 3

Next Steps

Track follow-ups and questions before exporting.

5 min

Wrap up the course with action items, links, or reminders. When you're ready, hit the export button to download all notes for this course as Markdown.

Page checklist

  • Include data-courseid on every form to scope exports.
  • Use unique data-sectionid per section so notes stay separate.
  • Add a hidden course-name input for the export filename.
  • Load noodle.js after the content so it can initialize forms.

Need a quick start?

  1. Copy a section block and adjust the IDs and titles.
  2. Update data-courseid to match your LMS course.
  3. Drop noodle.js onto the page.
  4. Test typing, auto-save, manual save, and export.
